## ðŸ§­ Curriculum Overview

### **Module 1: Introduction to Ethical Hacking and Penetration Testing**
**Focus:** Mindset, legality, and methodology  
- What ethical hacking is (and is not)
- Penetration testing vs vulnerability scanning
- Rules of engagement (ROE)
- Legal and ethical considerations
- Common penetration testing frameworks

**Deliverables:**
- Ethics & legality summary
- Penetration testing lifecycle diagram

---

### **Module 2: Planning and Scoping a Penetration Testing Assessment**
**Focus:** Before you touch the keyboard  
- Scoping targets and exclusions
- Defining objectives and risk tolerance
- Black-box vs gray-box vs white-box testing
- Authorization and documentation

**Deliverables:**
- Sample scope document
- Rules of Engagement (ROE) template

---

### **Module 3: Information Gathering and Vulnerability Scanning**
**Focus:** Reconnaissance wins wars  
- Passive vs active reconnaissance
- Network enumeration
- Service and version discovery
- Vulnerability scanning concepts and limitations

**Tools (examples):**
- Nmap
- OpenVAS / Nessus
- OSINT tooling

**Deliverables:**
- Recon notes
- Scan result analysis
- False-positive validation notes

---

### **Module 4: Social Engineering Attacks**
**Focus:** Humans are the softest target  
- Phishing concepts
- Pretexting and influence
- Physical and digital social engineering
- Defense and awareness strategies

**Deliverables:**
- Social engineering attack scenarios
- Risk and mitigation write-up

---

### **Module 5: Exploiting Wired and Wireless Networks**
**Focus:** Network-level compromise  
- Network attack fundamentals
- Password attacks
- Wireless security weaknesses
- Lateral movement concepts

**Tools (examples):**
- Metasploit
- Aircrack-ng
- Hydra

**Deliverables:**
- Exploitation walkthrough
- Evidence screenshots/logs

---

### **Module 6: Exploiting Application-Based Vulnerabilities**
**Focus:** Web applications are everywhere  
- OWASP Top 10
- Authentication and authorization flaws
- Injection vulnerabilities
- Input validation failures

**Tools (examples):**
- Burp Suite
- sqlmap
- Custom scripts

**Deliverables:**
- Vulnerability proof-of-concept (PoC)
- Business impact analysis

---

### **Module 7: Cloud, Mobile, and IoT Security**
**Focus:** Modern attack surfaces  
- Cloud shared responsibility model
- Common cloud misconfigurations
- Mobile app risks
- IoT threat landscape

**Deliverables:**
- Cloud misconfiguration assessment
- Risk summary and mitigation strategies

---

### **Module 8: Performing Post-Exploitation Techniques**
**Focus:** After access is gained  
- Privilege escalation
- Persistence techniques
- Credential harvesting
- Data exfiltration concepts
- Cleanup and evasion considerations

**Deliverables:**
- Post-exploitation notes
- Privilege escalation paths

---

### **Module 9: Reporting and Communication**
**Focus:** The report *is* the product  
- Executive vs technical reporting
- Risk ratings and impact
- Clear remediation guidance
- Communicating with stakeholders

**Deliverables:**
- Full penetration testing report
- Executive summary
- Remediation recommendations

---

### **Module 10: Tools and Code Analysis**
**Focus:** Understanding what tools actually do  
- Tool validation
- Script review and modification
- Custom tooling basics
- Interpreting exploit code safely

**Deliverables:**
- Tool comparison notes
- Script analysis documentation
